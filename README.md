Загрузчик файлов на Яндекс диск

Перед запуском:

```bash
npm install
```

Чтобы запустить:

```bash
npm run start
```

Для корректной работы приложения необходимо создать файл `.env` по примеру из `.env.example`
[Инструкция по получению токена](https://yandex.com/dev/disk-api/doc/en/concepts/quickstart)
